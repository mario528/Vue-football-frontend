## 技术栈

前端: Vue全家桶，Element-UI,Echarts.js,Socket.io

后端: Node.js,Express,Socket.io

数据库: MongoDB, Redis

## 项目运行

``` bash
# 克隆项目
git clone git@github.com:mario528/Vue-football-pro-frontend.git

# 进入前端项目根目录
npm install / cnpm install

# 确保后端服务开启的情况下
npm run dev

# 前端项目打包
npm run build

```
